# 시작하기

대시보드에 오신 것을 환영합니다! 이 가이드에서는 기본적인 사용법을 설명드리겠습니다.

## 주요 기능

대시보드에서는 다음과 같은 기능들을 제공합니다:

- **실시간 모니터링**: 시스템 상태를 실시간으로 확인
- **데이터 분석**: 다양한 차트와 그래프로 데이터 시각화
- **알림 관리**: 중요한 이벤트에 대한 알림 설정

## 빠른 시작

1. 좌측 메뉴에서 원하는 섹션을 선택하세요
2. 필요한 데이터를 확인하세요
3. 설정이 필요한 경우 설정 메뉴를 이용하세요

> **팁**: 각 섹션의 도움말을 참고하시면 더 자세한 정보를 확인할 수 있습니다.
