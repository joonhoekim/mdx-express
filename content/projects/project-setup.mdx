# 프로젝트 설정

새로운 프로젝트를 설정하는 단계별 가이드입니다.

## 1. 프로젝트 생성

새 프로젝트를 생성하려면:

1. **프로젝트** 메뉴로 이동
2. **새 프로젝트** 버튼 클릭
3. 프로젝트 정보 입력

## 2. 기본 설정

### 프로젝트 정보
- **이름**: 프로젝트의 고유한 이름
- **설명**: 프로젝트에 대한 간단한 설명
- **태그**: 분류를 위한 태그들

### 권한 설정
프로젝트 접근 권한을 설정할 수 있습니다:

| 역할 | 권한 |
|------|------|
| 소유자 | 모든 권한 |
| 편집자 | 읽기/쓰기 |
| 뷰어 | 읽기 전용 |

## 3. 고급 설정

### 환경 변수
```bash
# 개발 환경
NODE_ENV=development
API_URL=http://localhost:3000

# 프로덕션 환경
NODE_ENV=production
API_URL=https://api.production.com
```

### 배포 설정
자동 배포를 위한 설정을 할 수 있습니다:

- **Git 연동**: GitHub, GitLab 등
- **빌드 명령어**: `npm run build`
- **배포 대상**: Vercel, Netlify 등

## 마무리

설정이 완료되면 프로젝트 대시보드에서 진행 상황을 확인할 수 있습니다.
